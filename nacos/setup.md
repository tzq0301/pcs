# Nacos Setup Based on Docker

## 1. 拉取镜像

```shell
docker pull nacos/nacos-server
docker pull mysql
```

## 2. 创建 Docker 网桥

```shell
docker network create pcs-net
```

|-----|------------------------------------------------------------------|

## 3. 创建容器

### 3.1 创建 MySQL 容器

```shell
docker run -it --name pcs-nacos-mysql -e MYSQL_ROOT_PASSWORD=123456 -p 12001:3306 --network pcs-net --network-alias mysql -d mysql
```

### 3.2 创建 Nacos 容器

```shell
docker run -it --name pcs-nacos -e MODE=standalone -p 12000:8848 --network pcs-net --network-alias nacos -d nacos/nacos-server
```

## 4. 在 MySQL 中建表

### 4.1 进入 MySQL 容器

```shell
docker exec -it pcs-nacos-mysql bash
```

### 4.2 连接 MySQL

```shell
mysql -u root -p
```

输入密码 `123456`

### 4.3 创建数据库 `nacos`

```mysql
CREATE DATABASE nacos;

USE nacos;
```

## 5. 在 Nacos 中进行 MySQL 持久化配置

### 5.1 进入 Nacos 容器

```shell
docker exec -it pcs-nacos bash
```

### 5.2 编辑 conf/application.properties 文件

```shell
vim conf/application.properties
```

```properties
# 先进入 Nacos Server 容器，编辑 nacos 文件夹下的 conf 文件夹下的 application.properties 文件，并进行以下修改

spring.datasource.platform=mysql

# 172.18.0.2:3306 是我的 MySQL 的 IP 与端口号，nacos_mysql 是数据库名
# 使用 `docker inspect -f '{{range.NetworkSettings.Networks}}{{.IPAddress}}{{end}}' 容器id或容器name` 获得特定容器的 IP
db.url.0=jdbc:mysql://172.18.0.3:3306/nacos_mysql
# 注释掉 db.url.1
db.user=root
db.password=123456
```

### 5.3 退出 MySQL 与 MySQL 容器

```shell
[root@751e651c56fa nacos]# exit
exit
```
